Scene graph

A scene is a tree structure with a root and children. The tree consists of Object3D-instances. Each object has a transformation matrix.

Geometries

A geometry holds all data necessary to describe a 3D model. Example: Box, Dodocaheron. The structure of an object

Materials

Materials describe the appearance of objects: Is it shiny, is it green, does it have a texture image.

Mesh

In ThreeJS a Mesh is an Object3D consisting of a geometry and a material.


var mesh = new THREE.Mesh( geometry, material );
scene.add( mesh );

Lights

In graphics programming lights are strange abstractions. In practice the material of an object determines what fragment shader program to use. The shader program colors the pixels on the screen by calculating the color using information about the material and the lights in the scene. So a light is more of an abstract description of a light source.

Light calculations can be pretty complex but the simpler variants are called Lambert or Phong

Lambert shading is evaluated per vertex of the mesh. It is very fast but does not look very glossy.

Phong shading looks smoother since it is calculated per pixel. It can look 'shiny'.

Camera

A camera in this context does not simulate a physical. It is simply a mathematical description of the eye position and view direction.
